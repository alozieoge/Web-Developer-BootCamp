Overview
--------
- CSS = Cascading Style Sheets
- CSS is a language for describing how HTML documents appear - arranging and styling.
- It is used to style the visual representation of existing HTML docs.

CSS Rule / Pattern / Template
-----------------------------
selector {
  property: value;
}
